# Cidadania-Digital-e-Intelig-ncia-Artificial# Portal de Cidadania Digital: O Impacto das Deepfakes

### 📌 Objetivo do Projeto
Este site interativo foi desenvolvido como uma ferramenta de conscientização comunitária focado nos debates urgentes da atualidade tecnológica: o impacto das Deepfakes e o avanço da Desinformação Automatizada gerada por Inteligências Artificiais. Através de um jogo de memória interativo, o usuário aprende conceitos de verificação e fatos técnicos reais enquanto exercita a mente.

### 🚀 Tecnologias Utilizadas
* **HTML5 Semântico:** Uso de tags estruturais (`main`, `section`, `footer`) e formulários acessíveis.
* **CSS3 Avançado:** Layout totalmente maleável feito com Flexbox, uso de variáveis CSS e responsividade completa via Media Queries.
* **JavaScript Puro (Vanilla JS):** Manipulação dinâmica de elementos no DOM, controle de estados, contadores numéricos de tentativas e alternador de acessibilidade (Modo Escuro).

### 🤖 Engenharia de Prompts Aplicada (Ética Digital)
Para a concepção estética e estruturação da lógica desse portal, os seguintes comandos foram utilizados junto a IA:
1. *"Gere uma estrutura de jogo da memória usando apenas JavaScript sem bibliotecas externas, que manipule classes para exibição de texto."*
2. *"Desenvolva regras de layout CSS com Flexbox aplicáveis a grids de jogos que reduzam o tamanho proporcionalmente em telas móveis."*
